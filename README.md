# Proj-TMW

App de monitorizare Kg, talie, calorii si mese cu recomandari sanatoase
Mod -- slabit/crescut in kg -- gains
<p>Funct:</p>
Introducere la nivel de zi a noilor informatii

Trackere
KG
- Grafice evolutie kg -- bar char + line chart
- Semafor evolutie kg -- verde pt scadere, glaben pt constant, rosu pt crestere (sau invers dupa modul ales)
TALIE
- Grafice evolutie talie -- bar char + line chart
- Semafor evolutie talie -- verde pt scadere, glaben pt constant, rosu pt crestere (sau invers dupa modul ales)
Calorii
- Pie chart cu mesele, ce proportie au avut mesele din totalul zilei
- Tabel normal calorii per masa
